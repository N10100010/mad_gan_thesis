## DATA GENERATION

For all datasets (MNIST, FASHION-MNIST, CIFAR10) image have to be created, according to the following pseudo-code: 

```
models_for_data_generation = [madgan_1_gen, madgan_2_gen, ..., madgan_10_gen]
data_generation_by_model_by_gen = {}

for madgan_model in models_for_data_generation: 
    data_by_generator = {}
    for i, generator in enumerate(madgan_model.generators): 
        data_by_generator[i] = generator.create_images(100_000)
        
    data_generation_by_model_by_gen[madgan_model.__str__()] = data_by_generator

```
This results in a large amount of images by madgan by generator. All those images will be loaded and classified by an auxilary classifier, trained on the respective training dataset with traditional augmentations.

# Generative Data Augmentation

## Classifier - Statified Classifiers

For these experiments, a StratifiedDatasetCreated has been implement, that, given a path to generated images, can create a tf.keras.Dataset. The amount of real an generated images can be altered passing the respective ints during initialization. 

Classifiers are trained using data resulting from the above StratifiedDatasetCreator. For each dataset (MNIST; FASHION-MNIST; CIFAR10) and each model that generated labeled or unlabeled but classified images, the following classifiers will be trained and evaluated on the test/validation set: 

```
n_images_per_class = [ (0, 5000), (1000, 4000), (2000, 3000), (3000, 2000), (4000, 1000), (5000, 0)]
creation_experiments = []  # list containing all experiments, that created data and have labels

for creation_experiment in creation_experiments: 
    for n_real, n_fake in n_images_per_class: 
        Runner.run(
            StratifiedClassifierExperiment(
                epochs=50  # 50 for cifar, 35 for mnist and fashion
                n_real_images=n_real
                n_fake_images=n_fake
                dataset_name=cifar  # mnist, fashion, cifar
            )
        )

```
